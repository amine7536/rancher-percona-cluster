# Percona Ubuntu 14.04 Docker Image

This is a clone of the Docker [official image](https://docs.docker.com/docker-hub/official_repos/) for [percona](https://registry.hub.docker.com/_/percona/). See [the Docker Hub page](https://registry.hub.docker.com/pixelfactory/percona/) for the full readme on how to use this Docker image and for information regarding contributing and issues.

Docker official image is using Debian:jessie as the base image, this image is using Ubuntu Trusty instead.
